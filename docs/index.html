🔧 設定ベースカスタマイズ

JSONファイルでカテゴリ、フィールド、UIを完全カスタマイズ。コードを変更することなく、様々な用途に対応できます。

⚡ 高速検索

SQLite FTS5による高性能全文検索。リアルタイムで検索結果を表示し、快適なユーザー体験を提供します。

📊 多様な用途

ゲームアイテム、商品カタログ、ドキュメント管理、在庫管理など、幅広い用途に対応可能です。

🐳 簡単デプロイ

Docker対応で環境構築が簡単。一つのコマンドでアプリケーションを起動できます。
