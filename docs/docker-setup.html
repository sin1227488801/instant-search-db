🐳 Docker セットアップガイド

instant-search-db を最も簡単に起動するために

💡 なぜDockerが推奨？
Dockerを使うと、Python、依存関係、データベースなどすべてが自動でセットアップされ、 docker-compose up --build 一つのコマンドで起動できます。
🪟 Windows

Docker Desktop for Windows をインストールします。

📥 Docker Desktop をダウンロード
インストール手順:
  1. 上のリンクからインストーラーをダウンロード
  2. ダウンロードした Docker Desktop Installer.exe を実行
  3. インストール完了後、PCを再起動
  4. Docker Desktop を起動（デスクトップのアイコンから）
  5. 利用規約に同意してセットアップ完了
🍎 macOS

Docker Desktop for Mac をインストールします。

📥 Intel Mac 用ダウンロード 📥 Apple Silicon Mac 用ダウンロード
インストール手順:
  1. お使いのMacに対応するファイルをダウンロード
  2. Docker.dmg をダブルクリックして開く
  3. Docker アイコンを Applications フォルダにドラッグ
  4. Applications から Docker を起動
  5. 利用規約に同意してセットアップ完了
🐧 Linux (Ubuntu/Debian)

Docker Engine をコマンドラインでインストールします。

インストールコマンド: 
# 古いバージョンを削除
sudo apt-get remove docker docker-engine docker.io containerd runc

# 必要なパッケージをインストール
sudo apt-get update
sudo apt-get install ca-certificates curl gnupg lsb-release

# Docker公式GPGキーを追加
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# リポジトリを追加
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Dockerをインストール
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin

# 現在のユーザーをdockerグループに追加
sudo usermod -aG docker $USER

# ログアウト・ログインして設定を反映
✅ インストール確認方法
ターミナル（コマンドプロンプト）で以下を実行： 
docker --version
docker-compose --version
バージョン情報が表示されればインストール成功です！
🚀 インストール後の起動手順
git clone https://github.com/sin1227488801/instant-search-db.git
cd instant-search-db
docker-compose up --build
→ http://localhost:5000 でアクセス
